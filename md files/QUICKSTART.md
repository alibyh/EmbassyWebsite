# Quick Start Guide

## Installation & Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Production Build

```bash
# Build for production
npm run build

# Start production server
npm start
```

## Customization

- **Colors**: Edit `shared/config/colors.ts`
- **Typography**: Modify `shared/config/typography.ts`
- **Content**: Update data files in `entities/` and component files in `widgets/`
